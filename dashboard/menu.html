Menú Principal

Las opciones numeradas que el cliente ve al inicio o cuando no entiende algo

¿Cómo funciona esto?

El Menú Principal es lo primero que el cliente ve cuando escribe "hola", y también lo que aparece cuando el bot no entiende su mensaje. Es una lista numerada de opciones para que el cliente elija qué quiere saber.

Cliente escribe: "hola"→Bot muestra: 1️⃣ Preventa · 2️⃣ Actividades · 3️⃣ Instalaciones ...

¿Cómo fluye la conversación?

1Cliente escribe
→
2Bot muestra menú
→
3Cliente elige opción

Cada opción tiene un número y un título. El cliente puede responder con el número (1) o con palabras del título (Actividades). Cuando elige una opción, el bot responde con el mensaje que tú configures en Mensajes sueltos.

💡 Tip: aquí solo agregas, reordenas o borras opciones del menú. La respuesta que el bot da para cada opción se edita en Mensajes sueltos.
